## Download Tool

<Callout icon='ðŸ”—' type='info'>
  [Download CMWTAT
  Digital](https://github.com/TGSAN/CMWTAT_Digital_Edition/releases)
</Callout>

1. Click `CMWTAT_Digital_Release_2_5_0_0.exe` to download the tool.

![Download CMWTAT Digital](</static/images/blog/how-to-activate-windows-all-versions-for-free/step(1).png>)

2. After opening the tool, select your Windows version, here I am in the Professional version, so select `Professional`.
3. Click `Activate`

![Use CMWTAT Digital](</static/images/blog/how-to-activate-windows-all-versions-for-free/step(2-3).png>)

Wait a moment

![Remove old key](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(1).png>)

![Obtain permission](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(2).png>)

![Activating](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(3).png>)

Activate successfully

![Activated](</static/images/blog/how-to-activate-windows-all-versions-for-free/activating(4).png>)

## To check activation

Go to the Windows Settings activation page and you will see Windows is activated.

![Activated](/static/images/blog/how-to-activate-windows-all-versions-for-free/activated.png)

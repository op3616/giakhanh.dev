## Step 2 - Install VMware 16 Pro

1. Click Next

![Step 1](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(1).png>)

2. Accept Terms
3. Click Next

![Step 2-3](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(2-3).png>)

4. To enable enhanced keyboard support, please check Enhanced Keyboard Driver
5. Check Add VMware Workstation console tools into system PATH
6. Click Next

![Step 4-6](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(4-6).png>)

Please check the box according to your needs

1. Click Next

![Step 7](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(7).png>)

8. Click Next

![Step 8](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(8).png>)

9. Click Install

![Step 9](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(9).png>)

<Callout>Installation process, wait patiently</Callout>

![Installing](/static/images/blog/how-to-install-and-activate-vmware-16/installing.png)

1.  Click Finish

![Step 10](</static/images/blog/how-to-install-and-activate-vmware-16/step-2(10).png>)

This completes the installation of VMware Workstation Pro 16

## Bonus - Set up Traditional Chinese

<Callout icon='üîó' type='info'>
  [Download Traditional Chinese Language
  Pack](/static/files/how-to-install-and-activate-vmware-16/zh_TW.zip)
</Callout>

1. Right-click VMware Workstation Pro 16 and click Open File Location

![Step 1](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(1).png>)

2. Go to messages

![Step 2](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(2).png>)

3. Unzip the `zh-TW.zip` you just downloaded, then copy / move it to `messages`.

![Step 3](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(3).png>)

4. Right-click on VMware Workstation Pro 16 and click on Properties

![Step 4](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(4).png>)

5. Add `--locale zh_TW` at the end of "Target"
6. Press OK

![Step 5-6](</static/images/blog/how-to-install-and-activate-vmware-16/step-4(5-6).png>)

After opening VMware again, the interface successfully switched to Traditional Chinese
